# Prime Mind — Website

Static pre‑launch site for https://primemind.cloud

## Deploy on GitHub Pages
1. Create a public repo and add files in repo root.
2. Add `CNAME` with `primemind.cloud`.
3. Settings → Pages → Deploy from a branch → `main` / root.
